# 简历能力地图 · Resume Map

> 上传一份简历，自动生成「能力地图」：雷达图打分 + 真实能力 vs 缺口 + 面试风险点 + 闲时补强优先级。
> 把一次性的简历分析，沉淀成可复用的 AI 产品。

## 项目定位

面向**求职 / 转岗人群**（尤其 AI 产品、数据分析、财务数字化方向）的自助能力评估工具。
用户输入目标岗位 + 上传简历，系统输出一张可解释的能力地图，帮助用户：

- 看清自己**真实能力**与**岗位缺口**的差距
- 预判面试官会深挖的**风险点**（"看起来会、一问就露"的地方）
- 给出**闲时补强优先级**（P0 / P1 / P2）

## 当前状态

🧪 **原型阶段（V0）** —— 目前为静态单份分析的成品原型，评估逻辑由人工 + 模板驱动，数据均为可解释的结构化结果。

| 文件 | 说明 |
| --- | --- |
| `ability-map.html` | 单份简历能力地图（雷达图 + 风险点 + 补强路线），可交互查看 |
| `ai-competitor-radar.html` | 配套练手原型：AI 竞品雷达（高保真，含 PRD 设计说明页） |

## 产品规划路线

```
V0 规则原型    上传简历 → 预定义岗位维度 → 规则提取 → 雷达图        (当前)
V1 接 LLM      大模型做动态维度 + 有依据点评 + 风险点生成
V2 个性化      多份对比 / 职业路径规划 / 面试 Q&A 生成
V3 平台化      行业基准库 / 分享传播 / 招聘端 B 端能力 / 小程序
```

**落地方向**：先做 Web（零安装、好分享、PC 体验佳）；App 在验证期不做；小程序视微信传播需求再做。
**模型选型**：智能内核优先国产大模型（DeepSeek / 通义千问），要求中文好、结构化 JSON 输出、成本低、合规。
**合规红线**：简历为敏感个人信息（PII），必须本地处理 / 加密 / 不留存 / 可删除。
