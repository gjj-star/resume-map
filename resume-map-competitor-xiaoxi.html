竞品拆解 · 简历 / 招聘 AI 竞争格局

三款产品、两条轴线的错位竞争全景：小析智能（B 端中台）、飞职 AI 简历（C 端生成）、Resume Map（C 端评估教练）。同处"简历 → 结构化 → 能力解读"赛道，却各占一格。
拆解人：候选人（AI 产品经理方向）　数据基准：xiaoxizn.com / jianliai.com 官网（2026-07-15 抓取）　性质：作品集产出 / 产品拆解练习
一句话结论：三家同处简历 / 招聘 AI 赛道，但沿"用户侧（B 端 / C 端）× 价值层（生成 / 评估）"两条轴线完全错位。 小析智能是 B 端"读懂候选人"的中台； 飞职 AI 简历是 C 端"帮你写出漂亮简历"的生成器； Resume Map是 C 端"帮你看清能力缺口与补强路线"的教练。 飞职与我们用户侧重叠、威胁最大，但其终点是"一份简历文件"，我们的终点是"一份成长诊断"——这刀，是护城河。

一、竞品 A：小析智能（B 端招聘中台）

  • 定位B2B「AI 招聘中台」——"给招聘系统装上 AI 大脑"
  • 核心交付简历解析 API + 人岗匹配 + 候选人画像 + 知识图谱
  • 目标客户HR SaaS / ATS / 招聘平台 / 企业 HR / 猎头
  • 商业模式SaaS API（按量、1000+ 并发）+ 私有云
  • 六大能力简历解析 / 职位解析 / 候选人画像 / 人岗匹配 / 人才搜索 / 知识图谱
  • 官方指标解析 98% 准确率、200+ 字段、16 份/秒
  • 合规隐私每日清除、单份独立加密、可数据不出内网
  • 关键盲区"候选人画像"给 recruiter 用，不给求职者本人建议
指标为官网宣传口径，未经独立验证。详见本拆解 v1 版对小析的单家深挖。

二、竞品 B：飞职 AI 简历（C 端简历生成）

  • 定位免费 AI 简历生成工具——"0 经验也能做出专业简历"
  • 核心功能对话式生成简历、岗位技能匹配、深度优化、模板库
  • 目标用户求职者（应届生 / 转行 / 有经验 / 晋升）本人
  • 商业模式免费为主 + 增值；自称已服务 30w+ 求职者
  • 四步法进入对话 → 输入信息 → AI 生成 → 下载/编辑
  • 卖点命中 JD 关键词、成就量化、100% AI 筛选通过率（宣称）
  • 模板8 类（简约/商务/创意/文艺/应届/时尚/传统/科技）
  • 关键盲区只"写"不"诊"：产出简历文件，不告诉用户能力结构与缺口
"100% 通过 AI 筛选""80%+ 面试通过率"为宣传口径，未独立验证；免费模式下的真实转化与留存未知。

五、对 Resume Map 的启示（行动项）

1. 护城河锁定"评估 + 教练"层，两家竞品都不做

小析做解析、飞职做生成，无人做"诊断你差什么、怎么补"。把雷达图、风险点、P0/P1/P2 补强路线做深，就是别人抄不走的空位。

2. 飞职是最近邻威胁，需在它"加诊断"前卡位

飞职同为用户侧、已 30w 规模，若上线"能力评估"模块将正面撞我们。应对：1) 先做深诊断建立认知；2) 考虑"生成 + 评估"互补定位（飞职写、我们诊，可谈合作而非硬刚）；3) 强化其做不到的"多份对比 / 面试话术 / 职业路径"。

3. LLM 价值在"当教练"不在"当解析器 / 生成器"

解析有 小析、生成有 飞职，LLM 在我们手里应生成"有依据的能力点评 + 个性化补强建议 + 面试模拟问答"，避开两家的主战场。

4. 合规叙事两家的交点

小析强调"数据不出内网"、飞职有"个人信息安全" FAQ——C 端简历敏感，"本地处理 / 不留存 / 可删除"应成我们的信任背书（PRD 已写红线）。

反向对标一句话：小析做"企业视角的候选人画像"、飞职做"帮你写简历"、Resume Map 做"求职者视角的自我画像 + 补强教练"——同一份简历，三个相反 / 相补的方向。

六、风险与边界（诚实评估）

  • 飞职下探最接近我们的竞品，若加"能力诊断"模块将正面进入赛道，需持续监测其产品扩张。
  • 巨头压境招聘平台（Boss / 猎聘等）自带解析、匹配，若下探 C 端求职辅导，资源远超我们。
  • 免费挤压飞职以免费获客，个人端付费意愿本就弱，Resume Map 的免费 / 增值边界需谨慎设计。
  • 需求真实性三家印证赛道存在，但 C 端"为能力地图 / 诊断付费"意愿仍待 MVP 实测（PRD V0 核心假设）。
  • 信息局限仅基于两家官网首页，未申请试用，解析质量 / 真实定价 / 真实用户数未核实。
